# Agents Project Overview

## Purpose
A comprehensive production-ready system providing 67 focused, single-purpose plugins for Claude Code. This unified repository combines:

- **67 Focused Plugins** - Granular, single-purpose plugins optimized for minimal token usage and composability
- **99 Specialized Agents** - Domain experts with deep knowledge across architecture, languages, infrastructure, quality, data/AI, documentation, business operations, and SEO
- **107 Agent Skills** - Modular knowledge packages with progressive disclosure for specialized expertise
- **15 Workflow Orchestrators** - Multi-agent coordination systems for complex operations
- **71 Development Tools** - Optimized utilities including project scaffolding, security scanning, test automation, and infrastructure setup

## Key Features
- **Granular Plugin Architecture**: 67 focused plugins optimized for minimal token usage
- **Comprehensive Tooling**: 71 development tools
- **100% Agent Coverage**: All plugins include specialized agents
- **Agent Skills**: 107 specialized skills following progressive disclosure
- **Clear Organization**: 23 categories with 1-6 plugins each
- **Efficient Design**: Average 3.4 components per plugin

## Project Structure
- `.claude-plugin/` - Marketplace configuration (67 plugins)
- `plugins/` - 67 plugin directories, each containing:
  - `agents/` - Specialized agent definitions
  - `commands/` - Tools and workflows
  - `skills/` - Modular knowledge packages
- `docs/` - Comprehensive documentation:
  - `plugins.md` - Complete catalog of all 67 plugins
  - `agents.md` - All 99 agents organized by category
  - `agent-skills.md` - 107 specialized skills
  - `usage.md` - Commands, workflows, and best practices
  - `architecture.md` - Design principles and patterns
